
## Sapper


## Description 


A simple application / online game. The whole thing is based on firebase and allows you to connect two players to one game. 
Initially you enter your name / nickname and go to the panel where you have the opportunity to create your board with positions. It is possible to play the game created by other users. 

## Architecture

MVC (Model-View-Controller)

## Tech Stack

- Firebase
- Singleton Pattern 
- UI from Code 
- OOP (object-oriented programming)
- Clousures
